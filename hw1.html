Summary

In summary we learned about several important html processes. These included tables which allow us to order items into any table size with designated rows and columns. Additionally we learned about inline elements which allow us to edit text and elements within the actual html itself. These includes things like bold, italics, anchors which are links, and several others. Lastly, another important element we learned about were lists. If your webpage is in need of ordered and unordered lists, luckily html has you covered.
